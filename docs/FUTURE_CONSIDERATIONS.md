## 3. 성능/확장

### 샤딩 (Sharding)
- 대규모 데이터 처리
- 지역별 분산

- 현재 인메모리 → Redis로 전환
- 서버 재시작 시 캐시 유지

### CDN
- 프론트엔드 정적 자산 배포
- 글로벌 성능 향상
