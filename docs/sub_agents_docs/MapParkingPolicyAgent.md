# Map Marker & Parking Badge UX Specification

이 문서는 지역 기반 주차 정보 표시와 지도 마커/클릭 UX 정책을 정의합니다.

## 1. 지도 마커 표시

- **모든 후보 장소 마커는 지도에 항상 표시**
    - 지역 상태(OPEN, CANDIDATE, CORE)와 관계없이 지도에 마커 존재
    - CORE_REGION이 아니어도 마커는 클릭 가능
- **마커 색상/아이콘**
    - **CORE_REGION**: 기본 마커 + 주차 뱃지 표시
    - **OPEN/CANDIDATE**: 기본 마커만 표시, 뱃지 없음

## 2. 마커 클릭 시 동작

| 지역 상태 | 클릭 시 동작 | 표시 내용 |
| --- | --- | --- |
| **CORE_REGION** | Bottom Sheet 열기 | 장소 상세 정보 + **주차 뱃지** + 성공률/통계 |
| **OPEN_REGION / CANDIDATE_REGION** | Bottom Sheet 열기 | 장소 상세 정보 + **안내 문구**<br>“주차 정보는 실제 방문 기록을 기반으로 제공되며, 지역 및 장소에 따라 표시되지 않을 수 있습니다.”<br>**주차 뱃지/통계 미표시** |

🔑 **포인트**: 클릭 제한 없이 모든 마커 접근 가능, 단 상세 주차 정보는 CORE_REGION만 표시

## 3. 지도 초기 로딩

- **사용자의 현재 위치 근처를 지도 중심으로 로딩**
- CORE_REGION 전용 필터 또는 대구/경산 필터는 **UI에서 제거**
- 전체 후보 장소는 전국 범위에서 표시 가능
- CORE_REGION 전용 필터는 CORE_REGION 마커 강조 표시용으로만 사용

## 4. 마커 필터링

- **기본 필터**: 지역 전체 표시
- **선택적 필터**:
    - CORE_REGION만 강조 (마커 강조/배지 표시)
    - OPEN/CANDIDATE 지역은 마커 존재하지만 뱃지 없음
- **UI에서 “주차 뱃지 없는 곳은 숨김” 로직 금지**

## 5. 고정 문구 사용

안내 문구는 의미 변형 없이 그대로 사용:

> 주차 정보는 실제 방문 기록을 기반으로 제공되며,
> 지역 및 장소에 따라 표시되지 않을 수 있습니다.

**특정 지역명(대구, 경산 등) 삽입 금지**

## 6. 목표 UX 요약

1.  지도에 **전국 후보 장소 모두 표시**
2.  **CORE_REGION만** 주차 뱃지/통계 노출
3.  OPEN/CANDIDATE 지역은 클릭 시 **고정 안내 문구만 표시**
4.  필터링/클릭 제한으로 마커 접근 불가 상태는 **절대 없음**
